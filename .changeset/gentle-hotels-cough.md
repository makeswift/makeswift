'@makeswift/runtime': minor
---

Rebuilds the underlying mechanisms that are used to generate CSS style rules and manage `<style>` elements associated with Makeswift components. The new "CSS runtime" prevents unnecessary rerendering while editing in the Makeswift builder and drops several third party dependencies.

The runtime no longer utilizes an Emotion cache, and the signature of the `<RootStyleRegistry/>` component has changed accordingly. The `cacheKey` prop has been removed. We've introduced an optional `classNamePrefix` that can be used to set a custom prefix for all CSS class names generated by the Makeswift runtime. For example:

```typescript
export function MakeswiftProvider({
  children,
  locale = undefined,
  siteVersion,
}: {
  children: ReactNode
  locale?: string
  siteVersion: SiteVersion | null
}) {
  return (
    <ReactRuntimeProvider {...{ runtime, siteVersion, locale }}>
      <RootStyleRegistry classNamePrefix="foo-bar">{children}</RootStyleRegistry>
    </ReactRuntimeProvider>
  )
}
```
